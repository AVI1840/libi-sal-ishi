# Contributing to LIBI

תודה על העניין בתרומה לפרויקט LIBI.

## זה דמו אסטרטגי

LIBI הוא כרגע **דמו אסטרטגי** להצגה בפני גורמי מקצוע. תרומות מתקבלות בברכה, אך יש לשמור על עקרונות המנחים:

- לא לשבור לוגיקה עסקית קיימת
- לשמור על עברית כשפת ממשק ראשית
- לא להוסיף נתונים אישיים אמיתיים
- לשמור על פשטות — זה דמו, לא פרודקשן

## איך לתרום

### דיווח על בעיות
פתח Issue עם:
- תיאור הבעיה
- שלבים לשחזור
- סביבה (OS, Docker version)

### הצעת שיפורים
פתח Issue עם תג `enhancement` לפני שמתחילים לכתוב קוד.

### Pull Requests
1. Fork את הריפו
2. צור branch: `git checkout -b feature/your-feature`
3. ודא שהקוד עובד: `docker compose up -d`
4. פתח PR עם תיאור ברור

## קוד

- Python: עקוב אחר הסגנון הקיים (ruff, type hints)
- TypeScript: עקוב אחר הסגנון הקיים (ESLint, Prettier)
- אל תוסיף secrets או מידע אישי אמיתי

## שאלות

פתח Issue עם תג `question`.
